# O que é o pinup? É um utilitário simples de comunicação em rede. O programa cria uma conexão via socket na porta 30010 e permite que usuário troquem sinais de audio entre sí. Foi criado a muito tempo atrás apenas para testes e treino. Mas alguém achou utilidade para esse código. Portanto, resolvi deixa-lo público. # Posso usar o pinup comercialmente? Faça o que quiser.
